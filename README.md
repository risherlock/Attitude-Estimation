# Attitude-Estimation
Study and implementations of different attitude estimation algorithms for spacecrafts.

#### Implementations
1. [Multiplicative Extended Kalman Filter (MEKF) with Murrell's algorithm for spacecraft attitude estimation.](https://github.com/risherlock/Attitude-Estimation/tree/master/mekf_murrell)
2. [Crassidis, Markley - Unscented Filtering for Spacecraft Attitude Estimation (2003)](https://github.com/risherlock/Attitude-Estimation/tree/master/ukf_crassidis)
3. [Christian, Lightsey - Sequential Optimal Attitude Recursion Filter (2010)](https://github.com/risherlock/Attitude-Estimation/tree/master/soar)
4. [Anisvough, Zanetti - Q-Method Extended Kalman Filter (2017)](https://github.com/risherlock/Attitude-Estimation/tree/master/qekf)
